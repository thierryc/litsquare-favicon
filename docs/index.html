Litsquare Favicon

A portable workflow for turning your own Figma file, based on the Litsquare scaffold, into every required browser and app icon plus stable head tags.

Start GitHub

Quick start

Install the workflow in a project

1

Initialize

npm install --save-dev litsquare-favicon
npx litsquare-favicon init --framework react
2

Make your file

Open the litsquare favicon community scaffold, duplicate it, replace the visuals with your project artwork, hide guide layers, and export SVG frames into litsquare-favicon/source.

3

Generate

npx litsquare-favicon generate
npx litsquare-favicon validate

Figma source

Use the community file as a scaffold

The litsquare favicon Figma community file defines the frame names, safe areas, and export settings. Your project should export from its own copy after replacing the scaffold visuals.

Required export

Hide safe-area guide and maskable safe-area guide before export. If guide lines appear in generated output, re-export with those layers hidden.

  • favicon.svg: required 64x64 SVG source
  • icon.svg: optional 512x512 standard icon source
  • icon-maskable.svg: optional 512x512 maskable source
  • apple-touch-icon.svg: optional 180x180 Apple source
  • safari-pinned-tab.svg: optional monochrome mask
